# meme-maker
CLI, API and Slack bot to generate memes. Make memes not war.

**How to install?**
```
python setup.py install
```

**How to run?**
```
meme-maker [opts]
```

**Testing**
```
pip install -r requirements/test.py
python setup.py test
```
